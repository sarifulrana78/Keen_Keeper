Project Name: KeenKeeper—Your Own Shelf of Important Connections

Description: A clear explanation of how KeenKeeper helps users keep track of their interactions with friends, set relationship goals, see how they communicate, and build strong connections.


The technologies used are:

Next.js 16 and React 18
Recharts (for charts) and Tailwind CSS
Lucide React (symbols)
React Hot Toast (alerts)
TypeScript 
ESLint 

Three Key Features:
Friend Profile Management & Tracking — Detailed friend profiles with relationship goals and status tracking
Interaction Logging & Timeline — Log calls, texts, videos with complete history view
Friendship Analytics Dashboard — Interactive donut chart visualizing communication patterns with hover details
